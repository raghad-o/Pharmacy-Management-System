# Web Version - Pharmacy Management System

## Description
This is a front-end prototype of the Pharmacy Management System.

It provides the user interface design for:
- Login page
- Sign up page
- Main dashboard

---

## Features
- Clean UI design
- Navigation between pages
- Dashboard with medication actions (Add, Update, Delete, View)

---

## Note
This version is only a UI prototype and does not include backend functionality or database connection.
