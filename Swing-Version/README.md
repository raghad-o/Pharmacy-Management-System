# Swing Version - Pharmacy Management System

## Description
A desktop application built using Java Swing and JDBC to manage pharmacy operations with a MySQL database.

---

## Features

- User registration (Sign In)
- User login authentication
- Add medication
- Update medication
- Delete medication
- View medication records

---

## Database

Database name:
 `pms`

Tables:
- `login`
- `medicine`

---

## Technologies Used
- Java
- Swing
- AWT
- JDBC
- MySQL
