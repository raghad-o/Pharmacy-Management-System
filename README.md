# Pharmacy Management System

## Overview
This project is a Pharmacy Management System designed to manage medication data securely.  
It ensures that only authorized users can access and modify the system after login.

The system includes two implementations:

- Web Version (HTML/CSS frontend prototype)
- Desktop Version (Java Swing + JDBC + MySQL)

---

## Problem Statement
Pharmacies face challenges in managing medications due to their complexity and large number, making it difficult for pharmacists to easily update, delete, or add new medications.

---

## Features
- User Authentication (Login / Sign Up)
- Secure access after login
- Add medication
- Update medication
- Delete medication
- View medication records

---

## Technologies Used

### Web Version
- HTML5
- CSS3

### Desktop Version
- Java
- Swing
- AWT
- JDBC
- MySQL (XAMPP)
